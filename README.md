📊 HR Analytics Dashboard (Microsoft Excel – Project)
📌 Project Overview

This project features an interactive HR Analytics Dashboard built using Microsoft Excel.

The dashboard analyzes employee data to uncover insights related to workforce distribution, gender ratio, performance ratings, salary structure, leave tracking, skills distribution, age demographics, regional spread, and employment status.

The objective of this project is to demonstrate strong Excel skills in data analysis, dashboard design, KPI creation, and interactive reporting.

📊 Dataset Description

The dataset used in this project contains HR employee data with the following attributes:

Employee ID

Employee Name

Gender

Department

Job Title

Age

Region

Work Location (Head Office / Branch / Remote)

Employment Status (Full-time / Part-time / Contract)

Salary

Performance Rating

Leave Records

Skills

🎯 Business Insights

Workforce shows a balanced gender distribution (52% Male, 48% Female).

Central region has the highest employee concentration.

Majority of employees fall in the 26–35 age group.

Developers and HR Specialists contribute significantly to total salary expenses.

Communication and Management are dominant skill areas.

Employment structure shows a strong full-time workforce presence.
